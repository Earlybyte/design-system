# Modal Card

A Modal is a dialog that appears on top of the app's body, and must be dismissed by
the app before interaction can resume. In contrast to the [Modal](components/modal/readme.md) component, it is recomendet to use it whenever the content size is large.

  <span slot="head">Modal Title</span>
  <p>
    Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod
