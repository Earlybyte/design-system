# Footer

Basic footer for the Baloise Applications.

## Usage

The footer runs pure in blue or with a decorated track line below.

<docs-demo>
  <footer class="footer">
    <div class="container">Footer</div>
  </footer>
</docs-demo>

```html
<footer class="footer">
  <div class="container">Footer</div>
</footer>
```

### With Track Line

<docs-demo>
  <footer class="footer has-track-line">
    <div class="container">Footer</div>
  </footer>
</docs-demo>

```html
<footer class="footer has-track-line">
  <div class="container">Footer</div>
</footer>
```
