# Card Title

`bal-card-title` is a child component of `bal-card` that adds a title to card.

## Properties


| Attribute    | Description                                  | Type      | Default |
| :----------- | :------------------------------------------- | :-------- | :------ |
| **inverted** | If `true` the card text color becomes white. | `boolean` | `false` |
