## Properties


| Attribute      | Description                                                | Type      | Default |
| :------------- | :--------------------------------------------------------- | :-------- | :------ |
| **border**     | If `true` a bottom border is added to the data-item.       | `boolean` | `false` |
| **horizontal** | If `true` the data list is horizontal instead of vertical. | `boolean` | `false` |
