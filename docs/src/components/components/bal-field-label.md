### Properties


| Attribute    | Description                                        | Type      | Default |
| :----------- | :------------------------------------------------- | :-------- | :------ |
| **expanded** | If `true` the component takes the whole width      | `boolean` | `false` |
| **required** | If `true` a asterix (*) is added to the label text | `boolean` | `false` |
