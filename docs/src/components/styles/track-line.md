# Track Line

Decorated line used to separate header form the body.

## Usage

The track line is responsive and changes it size form `12px`, `14px` to `18px`

<docs-demo>
  <div class="bal-track-line"></div>
</docs-demo>

```html
<div class="bal-track-line"></div>
```

## Sizes
